FAQ



How to change the colour themes?
  • Open Up your Desired 'assets/color' file location. (For Example, 'assets/color/YOURCOLOUR.css'
  • Then Search For the Comment '<!-- Color CSS -->'
  • And remove the markup

Add: <link rel="stylesheet" href="assets/color/YOURCOLOUR.css">

<link rel="stylesheet" href="assets/color/theme-red.css">(Add this)

how to remove the top menu sticky as scrolling?

  • Open Up your HTML file 'cryptoking-HTML/' file location. (For Example, 'index.html'
  • Then Search For the Comment '<!-- START HEADER -->'
  • show this line - <header class="header_wrap fixed-top">
  • add class name - <header class="header_wrap fixed-top no-sticky"> (add class name 'no-sticky')
How can i add animation while scrolling to an item ?

You can do this, by using "animation" class and the class for the specific animation.

For example : <div class="animation" data-animation="zoomIn" data-animation-delay="1.6s"> >

- the "animation" class hooks the animation when the specific div is in viewport

- the "zoomIn" class is the animation effect. You can see the effects taht you can assing here : http://daneden.github.io/animate.css/

- the data-animation-delay="0.2s" is a variable that we can assing to define the time that the specific div displaying.

Where i can define my e-mail account to receive the messages from contact form?

  • Open Up your HTML file 'cryptoking-HTML' file location. (For Example, 'cryptoking-HTML/contact.php'
  • show this line number 11 - $mailTo = 'email@example.com'
  • 'email@example.com' enter your email account to receive the messages from contact form
